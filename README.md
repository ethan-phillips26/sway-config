# My Personal Sway Configuration

This is my personal Sway window manager configuration file. I use this to quickly get my keybinds on any machine running Sway.

_This repo does not contain any dot files. All styling must be done at your own risk._ \
<sub>(Risk being defined as "unappealing styling choices.")</sub>

## Packages

This config assumes these packages are installed:

* Waybar
* Alacritty
* Librewolf
